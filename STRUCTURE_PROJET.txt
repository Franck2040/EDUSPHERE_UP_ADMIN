## 🔧 ARCHITECTURE TECHNIQUE

### Frontend Stack
- **React 18** avec TypeScript
- **Vite** pour le build et dev server
- **TailwindCSS** pour le styling
- **Shadcn/UI** pour les composants
- **React Query** pour la gestion d'état et cache
- **React Router** pour la navigation
- **Lucide React** pour les icônes

### Backend & Base de données
- **Supabase** (PostgreSQL + Auth + Storage + Real-time)
- **Row Level Security (RLS)** pour la sécurité
- **Edge Functions** pour la logique métier
- **Policies** pour contrôle d'accès granulaire

### Sécurité & Permissions
```typescript
type AdminRole = 'super_admin' | 'moderator' | 'content_admin' | 'user';

// Permissions par rôle
super_admin: Accès total, création d'admins
moderator: Gestion utilisateurs, modération
content_admin: Gestion contenu, ateliers
user: Accès lecture seule
```

### Intégrations
- **Supabase Storage** - Upload et gestion fichiers
- **Supabase Real-time** - Notifications live
- **Supabase Auth** - Authentification sécurisée
- **React Query** - Cache intelligent et synchronisation
